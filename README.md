# ZSH Customization Theme
Custom Theme based on oh-my-zsh shell

### Installation

Simply clone this into a folder and edit the `~/.zshrc` file and add/edit the property

```
ZSH_CUSTOM=path/to/cloned/repo
```

### Features
Mac OS:

1. Battery Percent
1. RAM Usage
1. Timestamp

-----------------------
Linux:

1. RAM Usage
2. CPU Load
3. Timestamp
